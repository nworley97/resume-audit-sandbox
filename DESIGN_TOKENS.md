## üìã Usage Notes for Figma

1. **Color Format**: Use HEX values for most colors. OKLCH values are used for advanced color manipulation but can be converted to HEX for Figma.

2. **Spacing**: Always use multiples of 4px (the base unit).

3. **Typography**: 
   - Use Poppins for all headings
   - Use Inter or Geist for body text
   - Use SF Mono for code

4. **Border Radius**: Default is 8px, but use the scale provided for consistency.

5. **Shadows**: Use the shadow tokens for elevation. Avoid custom shadows.

6. **Dark Mode**: All colors have dark mode variants. Design both light and dark versions.

7. **Accessibility**: 
   - Text contrast should meet WCAG AA standards
   - Focus states use ring tokens
   - Interactive elements have hover states
